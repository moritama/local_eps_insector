# Local EPS Inspector

ブラウザ拡張機能として動作し、EPSファイルを完全ローカル（サーバー非依存）で高速に解析・プレビューするツールです。

## 特徴
- **完全ローカル処理**: ファイルをサーバーにアップロードせず、クライアントのブラウザ（拡張機能）内だけで完結するため、機密性の高いデザインデータでも安全です。
- **高速なメタデータ・プレビュー抽出**: 
  - ファイル先頭のテキスト情報（`%%Title`, `%%Creator`, `%%BoundingBox`等）を瞬時に抽出します。
  - DOS EPSヘッダーのバイナリ解析を行い、ファイル内に埋め込まれたプレビュー用TIFF画像を高速で抽出してCanvasに描画します。（UTIF.jsを使用）
  - 超巨大サイズのEPSでもブラウザをフリーズさせることなく処理が可能です。

## 使い方（Chrome拡張機能として）
1. `npm install` または `yarn` などで依存関係をインストールします。
2. `npm run build` を実行して `dist/` ディレクトリを生成します。
3. Chromeの拡張機能管理画面 (`chrome://extensions/`) を開きます。
4. 右上の「デベロッパーモード」をONにします。
5. 「パッケージ化されていない拡張機能を読み込む」をクリックし、生成された `dist/` フォルダを選択します。
6. 拡張機能のアイコンをクリックして現れるポップアップ領域に、EPSファイルをドラッグ＆ドロップしてください。

## 開発（ローカル動作）
```sh
npm install
npm run dev
```

## 注意事項
ブラウザ内で安全・高速に動作させるため、EPS（PostScript）自体の再レンダリングは行いません。プレビュー用画像が含まれないEPSの場合はネイティブソフト（Adobe Illustrator等）で開くよう促します。
